# webpack构建一个vue脚手架

webpac3.0+vue全家桶+node+express 构建一个脚手架

主要尝试使用node启动未打包的vue项目，且监听其文件变化

好处：在使用node为中间件的项目中，可以直接修改vue和node文件，不必每次调整node时还需将vue打包

主要使用插件：webpack-dev-middleware  webpack-hot-middleware

npm run dev

http://localhost:3000
